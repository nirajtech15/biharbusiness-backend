# 📖 Overview

BiharBusiness is a hyperlocal business discovery platform built to help businesses across Bihar increase their digital presence and generate leads.

The platform enables customers to discover businesses, contact them directly through WhatsApp, purchase premium listings, and search businesses by district and category.

The ecosystem consists of:

- 🌐 Website
- 📱 Android App
- 🔗 REST API
- 💳 Payment Gateway
- 📊 Admin Dashboard

# 🔌 API Highlights

| Method | Endpoint | Description |
|----------|----------------------|----------------|
| GET | /api/v1/businesses | Business List |
| GET | /api/v1/business/{slug} | Business Details |
| POST | /api/v1/auth/firebase-login | Login |
| POST | /api/v1/payment/create-order | Razorpay |
| POST | /api/v1/payment/webhook | Payment Verification |

# 🚀 Installation

```bash
git clone https://github.com/nirajtech15/biharbusiness-api.git

cd biharbusiness-api

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
